# Password Safe Tool
## Python 简易密码保管与随机密码生成工具
控制台轻量化账号密码管理工具，内置随机高强度密码生成功能，数据本地文本持久化保存，练习随机库、文件读写、字典增删改查、字符串操作。

## 项目功能
1. 随机密码生成：自定义长度，包含大小写字母、数字、特殊符号，安全性高
2. 账号存储：录入网站、账号、密码，自动写入 `pwd.txt`
3. 信息查询：输入网站名快速读取账号密码
4. 数据修改：单独更新账号或密码，无需重新录入全部信息
5. 记录删除：清除不需要的网站账号信息
6. 容错校验：长度限制、空值拦截、非数字输入捕获，程序稳定不闪退
7. 免手动输密码：新增记录时回车可自动生成12位密码

## 运行说明
1. 源码保存为 `password_safe.py`，运行后自动生成 `pwd.txt` 存储数据
2. 菜单数字选择功能，0退出自动保存所有记录

## 核心学习知识点
1. random 随机模块：sample 随机抽取字符生成密码
2. string 字符常量：大小写字母、数字集合
3. 文件读写 with 上下文管理器，分行读写、分隔符分割数据
4. 字典存储多维数据：key=网站名，value=[账号,密码]
5. while True 菜单循环、break 退出程序
6. try-except 捕获数值输入异常
7. 字符串判空、拼接、分割、strip清理换行空格
